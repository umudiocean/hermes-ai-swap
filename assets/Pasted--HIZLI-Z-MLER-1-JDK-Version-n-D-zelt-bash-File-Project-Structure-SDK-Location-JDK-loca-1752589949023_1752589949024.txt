🔧 HIZLI ÇÖZÜMLER
1. JDK Version'ını Düzelt
bash# File > Project Structure > SDK Location
# JDK location'ı kontrol et ve değiştir
# Önerilen: JDK 17 veya JDK 11 kullan
2. Gradle Cache Temizle
bash# Android Studio Terminal'de:
./gradlew clean
./gradlew cleanBuildCache

# Veya manuel olarak:
# C:\Users\Matt\.gradle\caches klasörünü sil
3. Android SDK Manager'da Güncellemeler
bash# Tools > SDK Manager
# SDK Tools sekmesinde:
✅ Android SDK Build-Tools (latest)
✅ Android SDK Platform-Tools
✅ Android SDK Tools
# Update/Install yap
4. build.gradle Dosyasını Güncelle
gradle// android/build.gradle
android {
    compileSdk 34
    
    compileOptions {
        sourceCompatibility JavaVersion.VERSION_1_8
        targetCompatibility JavaVersion.VERSION_1_8
    }
    
    // Bu satırı ekle:
    packagingOptions {
        pickFirst '**/libc++_shared.so'
        pickFirst '**/libjsc.so'
    }
}
5. gradle.properties Dosyasına Ekle
properties# android/gradle.properties
org.gradle.jvmargs=-Xmx2048m -Dfile.encoding=UTF-8
org.gradle.parallel=true
org.gradle.daemon=true
org.gradle.configureondemand=true

# Bu satırı ekle:
android.enableJetifier=true
android.useAndroidX=true
🚨 ADVANCED ÇÖZÜMLER
6. JDK Path'i Manuel Ayarla
bash# File > Settings > Build, Execution, Deployment > Build Tools > Gradle
# Gradle JVM: Use Project JDK seç
# Veya manuel path ver: C:\Program Files\Java\jdk-17\
7. Android Studio'yu Yeniden Başlat
bash# 1. Android Studio'yu kapat
# 2. Task Manager'da tüm java.exe process'lerini sonlandır
# 3. Android Studio'yu yeniden aç
# 4. Invalidate Caches and Restart yap
8. Flutter Doctor Komutu
bash# Terminal'de:
flutter doctor -v
flutter clean
flutter pub get
🎯 EN ETKİLİ ÇÖZÜM
Android Studio > File > Invalidate Caches and Restart

"Invalidate Caches and Restart" seç
"Invalidate and Restart" butonuna tıkla
Yeniden başladıktan sonra:

bashflutter clean
flutter pub get
flutter run
📱 FLUTTER SPECIFIC
Eğer Flutter kullanıyorsanız:
bash# pubspec.yaml'da environment kısmını kontrol et:
environment:
  sdk: ">=3.0.0 <4.0.0"
  flutter: ">=3.10.0"

# Sonra:
flutter upgrade
flutter pub get
🔍 DEBUGGING
Hâlâ çalışmıyorsa:
bash# Detaylı log için:
flutter run --verbose

# Veya Android Studio'da:
# Run > Edit Configurations
# Additional arguments: --verbose
